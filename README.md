# Comprehensive-Design
63组综设代码管理
大家更新代码前记得先pull再push，不然会冲突



| 位置                  | 建议内容   | 说明                          |
| :-------------------- | :--------- | :---------------------------- |
| `./assets/images/`    | 图片资源   | 界面截图、图标、设计素材等    |
| `./assets/videos/`    | 视频资源   | 演示视频、教程录像等          |
| `./assets/templates/` | 模板文件   | 项目模板、配置模板等          |
| `./docs/`             | 项目文档   | 设计文档、用户手册、API文档等 |
| `./releases/`         | 发布版本   | 各版本的可执行程序打包文件    |
| `./src/firmware/`     | 下位机源码 | 嵌入式固件、硬件相关代码      |
| `./src/software/`     | 上位机源码 | 桌面应用程序、软件代码        |
